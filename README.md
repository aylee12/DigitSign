# DigitSign
전자서명 웹 사이트 <br>
Java -> Java + JSP로 리팩토링 <br>

## 기능 <br>
1. 키 생성 <br>
2. 전자서명 생성 <br>
3. 전자서명검증 <br>
키 생성에는 SHA1withRSA 알고리즘 적용 <br><br>

## 사용 기술 <br>
Java, JSP <br><br>
